How to Use Paint-a-Sketch 98

Controls:

  • ARROW KEYS: Move the cursor & draw lines
  • CLICK: Jump cursor & draw line to click position
  • S KEY: Erases it by "shaking" (My excuse to use animation)
  • C KEY: Change drawing color

Tools:

  • Cursor (Selected): Draw with arrow keys or click
  • Other tools: For decoration only (making the classic Paint vibe complete.)

Tips:

  • Keep cursor within the canvas area
  • Try making geometric patterns!
  • Click the color boxes to change line color or use the C KEY
Windows 98 Tip: For more help, either contact me directly or try to look in the Paint for Windows 98's user manual. Good luck finding that manual.
